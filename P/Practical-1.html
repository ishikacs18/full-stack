About Course

Front-end web development is everything involving in programming the user interface of a web application.
Typically it refers to the hyper text markup language(HTML), Cascading Style Sheet(CSS) and Java Script(JS) portion of a website as opposed to the database or server-side programming. It encompasses everything from building a simple page of HTML text to creating complex, responsive HTML5 website designed to be accesses via various different browser, devices and screen sizes.
